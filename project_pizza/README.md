🍕 Pizza Sales Analysis Dashboard

Project Overview  
This Power BI dashboard analyzes pizza sales performance and provides insights into revenue, order trends, and best-selling pizzas.

Tools Used  
- Power BI  
- SQL  
- Excel  

Key KPIs  
- Total Revenue  
- Total Orders  
- Average Order Value  
- Total Pizzas Sold  

Insights  
- Highest sales on weekends  
- Classic category generated maximum revenue  
- Large-sized pizzas contributed the most sales  
- Top-selling pizza identified using sales volume
